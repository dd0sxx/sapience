# Model Context Protocol Server

Use Sapience's API with any [MCP client](https://modelcontextprotocol.io/clients) via the HTTP endpoint:

```
https://api.sapience.xyz/mcp
```

* [**Claude**](/mcp/use-with-claude) - Integrate Sapience with [Claude](https://claude.ai) to provide it with tools that query the Sapience GraphQL API for live data, prepare transaction data for prediction markets, and more.
* [**Cursor**](/mcp/use-with-cursor) - Allow agents in [Cursor](https://www.cursor.com/) to query the Sapience API directly, helping it better assist with software development.
* [**Langchain**](/mcp/use-with-langchain) - Build research agents that trade prediction markets using [LangChain](https://www.langchain.com/) with Python or TypeScript.
* [**Google ADK**](/mcp/use-with-google-adk) - Build research agents that trade prediction markets using [Google's Agent Development Kit](https://google.github.io/adk-docs/).
