### Tutorial

1. **Choose a vault or market** to support.
2. **Deposit USDe** into the pool.
3. **Provide liquidity** manually or via open-source bots.
4. **Earn yield** from fees and USDe’s native reward rate.
5. **Withdraw** after settlement or when you want to reallocate.
