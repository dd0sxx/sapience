## Use Sapience Docs in Cursor

To add Sapience documentation to [Cursor](https://cursor.com) for contextual AI assistance:

1. In Cursor, type `@Docs` in the editor or chat
2. Select `Add new doc`
3. Enter `https://sapience.docs.xyz/` as the URL (include the trailing slash to index all subpages)
4. Provide a name for the documentation (e.g., "Sapience Docs")
5. Click "Index" to add the documentation to your context

Once indexed, you can reference Sapience documentation in your queries using `@Docs Sapience Docs`.

You can manage your added documentation under Cursor Settings > Features > Docs.
